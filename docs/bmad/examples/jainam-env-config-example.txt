# ============================================================================
# Jainam Symphony Multi-Server Configuration Example
# ============================================================================
# This example shows the complete configuration pattern for all Symphony 
# servers (A, B, C) and both dealer account types (PRO, NORMAL).
#
# USAGE:
# 1. Copy the configuration block for your active server/account to your .env
# 2. Set JAINAM_ACTIVE_SYMPHONY_SERVER to your server (A, B, or C)
# 3. Set JAINAM_ACTIVE_ACCOUNT_TYPE to your account (PRO or NORMAL)
# 4. Fill in your actual credentials (replace example values)
# ============================================================================
